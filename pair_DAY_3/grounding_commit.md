# Grounding Commit — Day 3

## What Changed

**File edited:** `methodology.md` in `tenacious_bench`

**Specific edit:** The paragraph presenting the 82% capacity_honesty score was revised to scope the claim to in-distribution performance and document what the eval data actually tests.

## Before

```
The SimPO adapter achieves 82% capacity_honesty on the held-out eval set,
compared to 71% for the baseline prompt-only condition — a 15.5% relative
improvement. This result is Pareto-optimal: the adapter matches the constrained
prompt on capacity_honesty while costing 73% less per task.
```

## After

```
The SimPO adapter achieves 82% capacity_honesty on the held-out eval set,
compared to 71% for the baseline prompt-only condition — a 15.5% relative
improvement. This eval is in-distribution: all 200 training pairs and the
eval set were generated by the same programmatic_template_v1 pipeline, meaning
chosen and rejected responses share vocabulary and escalation patterns across
both splits. The Pareto-optimality claim holds within this template distribution.
Whether the preference generalizes to novel phrasing — prospects who express
commitment pressure in language outside the template pool — is untested. A
paraphrase robustness eval is planned for v2 training data: the same 82 unique
input contexts rewritten so escalation and hard-commitment are expressed with
different words, holding semantic content constant. If accuracy holds under
paraphrase, the adapter learned the behavioral rule. If it drops, the 82%
figure reflects template vocabulary matching rather than generalized preference
learning.
```

## Why This Edit

The original paragraph presented 82% as a deployment-ready number without stating what the eval was actually measuring. The paraphrase distinction matters because the adapter will encounter prospect replies that use none of the template vocabulary — that is the realistic deployment case. Bounding the claim to in-distribution performance and naming the planned robustness check gives any reader a precise account of what has been tested and what has not, which is the minimum required to defend the Pareto-optimality argument honestly.
